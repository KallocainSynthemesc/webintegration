La Kallocaïne

La Kallocaïne ou Kallocaïne1 (en suédois Kallocain) est un roman dystopique suédois publié en 1940 par Karin Boye. Le roman figure parmi les classiques du genre de la dystopie, qui s'est développé entre les deux guerres mondiales avec le concept de « totalitarisme », rapprochement fait par Hannah Arendt entre le communisme et le nazisme
